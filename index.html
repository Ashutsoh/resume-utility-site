Why Does Password Strength Matter?

A weak password (like password123) can be cracked by automated tools in under 1 second. A strong password with 12+ mixed characters can take trillions of years to brute-force. This matters for your SBI Net Banking, Gmail, UPI apps, and social media.

⚠️ Common Mistakes Indians Make: Using mobile numbers (9876XXXXXX), DOB (01011990), city names, or Bollywood references. These are in every hacker's dictionary list.

What Makes a Good Generated Password?

Length is king. A 16-character random password is astronomically stronger than a 10-character one. For banking accounts, use 16+ characters. For social media, 12+ is fine. Use a different password for every site — never reuse them.

Passphrase vs Password — Which Is Better?

A passphrase like correct-horse-battery-staple is both memorable and strong. A random password like xK9#mP2!rL is slightly stronger per character but harder to memorise. Use passphrases for accounts you type frequently.

What is a Dictionary Attack?

Hackers use automated programs that try millions of common passwords per second. Lists include English words, names, Bollywood references, number patterns (123456, 111111), and passwords leaked in previous data breaches (Facebook, Zomato, BigBasket, etc.).

General Security Tips for Indian Users

Never reuse passwords across banking, email and social media. If one is breached, all are compromised.
UPI fraud alert: You don't enter a PIN to receive money. Any request to "verify" with your PIN is a scam.
Phishing is #1 threat in India. Always check the sender's actual email address — not just the display name.
Use a password manager (Bitwarden is free and open-source) to store unique passwords for every site.
CERT-IN (India's cyber security agency) recommends 2FA on all critical accounts. It stops 99% of automated attacks.
